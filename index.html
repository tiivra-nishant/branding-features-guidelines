Branding Guidelines

The following are the branding guidelines for the online website of Tiivra (https://www.tiivra.com).

Logo & Favicon

The logo should not be altered by any means. The original logo is present in a PNG format and is given below.

Changes to the Tiivra logo (including, but not limited to, rotating it, changing its transparency, changing its colors) are not allowed.

A favicon is a small image that shows up on the browser tab left to the title of the webpage. TIIVRA's favicon should be used as is without any alteration to its style. Like the original logo for the site, the image is located in the Files section.

Headings

There are a total of six heading sizes available (ranging from <h1> through <h6>), with each heading having specific font sizes and weight. The font family used is TT-Supermolot, and is uploaded to the Files section of the site's Shopify theme.

Heading 1

The primary heading size. Should be 42px in size. Font weight: 900. Follows the same weight for all heading sizes.

Heading 2

The secondary heading size. Should be 32px in size.

Heading 3

The tertiary heading size. Should be 18px points in size.

Heading 4

The tertiary heading size. Should be 18px points in size.

Heading 5

The tertiary heading size. Should be 16px points in size.

Heading 6

The tertiary heading size. Should be 16px points in size.

Body

Body and paragraph font size should be 16px, with a line height of 1.5. The font family used here is Publica Sans Round, and similar to the heading font family, is uploaded to the Assets directory of the site's Shopify theme.

We should avoid using too many different fonts to ensure that the website isn't cluttered and to keep UX principles in mind. Additionally, excess fonts increase requests made to external files and if the fonts are not available in the user's systems (high likelyhood, since these are external fonts available primarily through the web), the browser will have to first the typeface locally onto the user's device.

Colors

The following colors are specified to be Tiivra's primary and secondary colors. The Hex, RGB, and CMYK values are listed below.

  • Primary

    Hex: #f8cb23

    RGB: (248, 203, 35)

    CMYK: (0, 18, 86, 3)

  • Secondary

    Hex: #000000

    RGB: (0, 0, 0)

    CMYK: (0,0,0,1)

Buttons

The primary call-to-action (CTA) button should have a solid background that adheres to the primary color of Tiivra's brand. The secondary CTA button should have an outline that has the same primary color of Tiivra, but the background this time should be a solid color that matches the background of the respective section.
